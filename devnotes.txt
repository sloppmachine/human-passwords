Error message format: "Error: (description)" -> immediate exit

save file structure:
- 4 bytes: amount of words contained in the pool 
- length of the huffman tree section (the amount of nodes)
- for each leaf of the huffman tree: 1 byte for the char, 1 byte for the length of the code, then the code (using bits)
- 1 byte: amount of included bits in the last byte of the stream of words (the included bits are more significant than the not included ones)
- stream of words to be read using the huffman tree (using a self-chosen seperator)
- EOF
